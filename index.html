Arktoolbox

(The Autonomous Robotics Kit Scicoslab Toolbox)

The goal of this project is to provide simulation tools for guidance, control, and navigation algorithm development for micro aerial vehicles.

Arktoolbox currently works with Scicoslab on Debian GNU/Linux, Mac OS X, and Windows. Support for Scilab is in development.
